# E-Commerce Sales Analytics (MySQL)

## Project Description
This project analyzes sales data from a simulated e-commerce platform using MySQL.
It focuses on extracting business insights such as revenue trends, top customers,
product performance, and customer retention.

## Tech Stack
- MySQL
- SQL

## Key Business Questions Answered
- What is the monthly revenue trend?
- Who are the top 10 customers by spending?
- Which products generate the highest revenue?
- What is the customer repeat purchase rate?

## Database Tables
- customers
- products
- orders
- order_items
- payments

## Skills Demonstrated
- Complex JOINs
- CTEs & Window Functions
- Time-based analysis
- Business-focused SQL queries
